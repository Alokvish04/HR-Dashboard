# HR-Dashboard
A Power BI dashboard designed to visualize key HR metrics, including attrition trends, employee demographics, and headcount distribution. This project leverages DAX and data modeling to transform raw workforce data into actionable organizational insights.

# Key Features
**Attrition Analysis:** Visualizes turnover patterns and identifies factors contributing to employee departures.
**Demographic Insights:** Tracks workforce composition including gender, age groups, and departmental distribution.
**Performance Monitoring:** Provides a high-level view of organizational health and employee metrics.
**Interactive Filtering:** Allows users to drill down into specific data segments using dynamic slicers and cross-filtering.

# Technical Details
**Data Modeling:** Established relationships between multiple tables to ensure a cohesive star schema
**DAX (Data Analysis Expressions):** Implemented custom measures for complex calculations such as attrition rates and average tenure.
**UI/UX Design:** Utilized custom JSON themes for consistent branding and visual clarity.
**File Format:** The project is contaimed within a .pbix file, ready for deployment in Power BI Service or Desktop.

# How to Use
**Clone the Repository:** Download the project files to your local machine.
**Open in Power BI Desktop:** Ensure you have the latest version of Power BI Desktop installed to view the .pbix file.
**Explore Data:** Use the interactive elements on the report canvas to filter by department, job role, or timeframe.

# Future Enhancements
**Predictive Analytics:** Integrating machine learning models to forecast future attrition trends.
**Live Data Integration:** Connecting the dashboard to real-time HRIS APIs for automated updates.
